## Common Launch Day Issues

### Technical
**Problem:** Game won't download
- Check build is set to default branch
- Verify depot configuration
- Test from another account

**Problem:** Crashes on startup
- Hot fix immediately
- Post announcement
- Provide workarounds
- Push patch ASAP

**Problem:** Performance issues
- Gather system specs from reporters
- Identify common thread
- Provide temporary solutions
- Schedule optimization patch

### Store Page
**Problem:** Wrong price showing
- Check regional pricing settings
- Verify currency conversions
- Update if needed

**Problem:** Missing features
- Check depot includes all files
- Verify DLC is enabled
- Test purchase flow

### Community
**Problem:** Review bombing
- Identify cause
- Address issue
- Communicate fixes
- Stay professional

**Problem:** Negative press
- Assess validity
- Respond appropriately
- Fix real issues
- Learn from feedback
